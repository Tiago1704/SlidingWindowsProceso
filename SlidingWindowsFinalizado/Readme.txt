Los audios que se analizarán deben estar en la misma carpeta que el main.

La función importante es SliWin, ella hace todo el procedimiento.
Hay algunas extras agregadas, esas son:

MuestraDeVectores: Muestra el arreglo de cada audio.
ComparacionDeVectores: Compara cosas extras, como la frecuencia, o el tamaño.
MostrandoArrays: Solamente Muestra los arrays.
